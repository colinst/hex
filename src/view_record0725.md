数据脏写怎么办？
```
缓存里数据1被Server process A修改了,   那么缓存里的数据1已经与数据文件中的数据不一致了,那么数据1这时就成为了脏数据,  那么Server process B就不能读出那个脏的数据1了, 跟数据库里的不一致嘛, 否则就形成了脏读.
```
避免脏写脏读
```
缓存重新写一次，或者读的时候触发缓存
```
